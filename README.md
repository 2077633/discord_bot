# discord_bot
Discord bot qui detecte quand un user join un chat vocal dans mon discord.

Envoie une notification discord au user_id spécifié.

# Lancement

dans wsl : python bot.py

ou

executer le fichier .bat

# Token

Pour avoir le token : 

https://discord.com/developers/applications/1452776706342518987/bot

# User Id

Pour avoir le user id : 

Activer le discord developer

Clic droit sur votre porfil, "identifiant"

# Lancer le script au démarrage

win + r "shell:startup"

mettre un raccourci du .bat
